Environmental Dataset Gateway Data Download Metrics

The Environmental Dataset Gateway (EDG) serves as the Agency's enterprise metadata catalog for datasets produced by the U.S.EPA. The EDG Data Download Location provides a download site for Region, Program, and Laboratory data managers to share their datasets. Metadata records cataloged in EDG direct users to the data download location for that resource. A LAN ID and password is required to access restricted datasets.

This site contains annual and monthly aggregate download statistics for files hosted on the EDG Data Download Location. These statistics are organized by access level (public vs. restricted), program (subfolders) and dataset. Downloads can be viewed for a specific month and year, using dropdowns menus. All statistics are updated nightly. EPA data mangers can use this tool to track downloads of their data for use by EPA and the public.
